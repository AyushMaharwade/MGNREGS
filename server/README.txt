Run the server
1) Install deps: npm install
2) Set env vars:
   - PORT (default 8080)
   - DATA_GOV_API_KEY
   - DATA_GOV_RESOURCE_ID
3) Start: npm start

Endpoint
- GET /api/mgnregs/chhattisgarh/monthly?district=Raipur&month=01&year=2025
  Returns placeholder JSON until you wire data.gov.in in index.js.
